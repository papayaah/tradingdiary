# Repository Instructions

## Git commits

- Do not add AI, Codex, agent, assistant, or automated-tool attribution to commits.
- Do not add `Co-authored-by` trailers for AI or automated tools.
- Do not add “generated by,” “assisted by,” or similar attribution to commit messages, pull requests, or release notes.
- Use only the repository user's configured Git author and committer identity.

## Branches

- Continue working on the currently checked-out branch unless the user explicitly asks for a new branch.
- Do not create `agent/*`, `codex/*`, or similarly attributed branches unless the user explicitly requests one.
